# Print a MOAT audit summary

Print a MOAT audit summary

## Usage

``` r
# S3 method for class 'moat_audit'
summary(object, verbose = FALSE, ...)
```

## Arguments

- object:

  A `moat_audit` object.

- verbose:

  A single logical value. When `TRUE`, include all module reasons and
  recommendations. Defaults to `FALSE`.

- ...:

  Additional arguments passed to methods.

## Value

A `summary.moat_audit` object.
